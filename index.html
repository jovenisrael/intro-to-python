Introduction to Python

A friendly, step-by-step guide for students with no programming background.

1. Introduction to Python

What Python is

Python is a programming language. A programming language is a way to give instructions to a computer using words and symbols that humans can read.

Think of Python as a friendly translator between you and the computer. You describe what you want, and Python turns it into steps the computer can follow. 

print("Hello, world!")

This one line tells Python to show a message on the screen.

Where Python is used in real life

Python is used for websites, data analysis, automation (like sorting files), games, and even controlling robots.

  • Companies use Python to analyze sales and customer data.
  • Students use Python to build simple games or calculators.
  • Offices use Python to automate repetitive tasks.

Why Python is beginner-friendly

Python looks close to everyday English. It does not require lots of extra symbols, so beginners can focus on ideas instead of confusing punctuation.

You can also test code quickly, make changes, and see results right away. This helps you learn by trying small examples.

How Python code is read (top to bottom)

Python runs your instructions from top to bottom, line by line. The order matters, just like following steps in a recipe. 

print("Step 1")
print("Step 2")
print("Step 3")

If you move lines around, the output will change. Always write code in the order you want it to run.

Comments (notes for humans)

Comments are notes you write for yourself. Python ignores them, but they help you remember what your code does. 

# This is a comment
print("Python ignores the comment")

2. Variables & Data Types

What a variable is (real-world analogy)

A variable is like a labeled box. The label is the variable name, and the box holds a value. You can look inside the box or replace what is inside.

For example, a box labeled age can hold the number 20. Later, you can replace it with 21 when the birthday comes.

How assignment works

Assignment means putting a value into a variable using the equals sign =. This does not mean “equal”; it means “store.” 

score = 10
score = 15
print(score)

The final value is 15 because the second assignment replaces the first one. 

points = 3
points = points + 2
print(points)

You can also build new values from old ones.

Variable naming rules

  • Use letters, numbers, and underscores.
  • Start with a letter (not a number).
  • Use descriptive names like total_price.

Data types

  • int: whole numbers like 1, 2, 10
  • float: numbers with decimals like 3.5, 99.5
  • str: text in quotes, like "Hello"
  • bool: true/false values, True or False

Data types matter because Python needs to know how to handle each value. You can add numbers, but you cannot add text and numbers without converting them first.

Simple examples

age = 20
price = 99.5
name = "Alex"
is_student = True
print(age + 5)
print(name + " Smith")
print(is_student)

Strings (text) in more detail

Strings are text in quotes. You can combine them and check their length. 

message = "Hello"
print(message + "!")
print(len(message))

Useful beginner string methods:

  • upper(): make text uppercase
  • lower(): make text lowercase
  • replace(): swap part of the text
word = "Python"
print(word.upper())
print(word.lower())
print(word.replace("Py", "My"))

Booleans (True/False)

Booleans are used for decisions. Comparisons create boolean values. 

is_tall = True
is_short = False
print(5 > 3)
print(2 == 10)

Built-in functions

print() shows a value on the screen so you can see it.

type() tells you the data type of a value. 

print(name)
print(type(price))
print(type("Hello"))
print(type(100))
print(type(3.14))
print(type(False))

More useful beginner functions:

  • len(): how many items are in a list or how many characters in text
  • int() and float(): convert to numbers
  • str(): convert a value to text
print(len("Python"))
print(str(42))
print(int("7") + 3)

3. Input & Output

Getting input

input() lets the user type something. Python stores it as text (a string). 

city = input("Enter your city: ")
print("You live in", city)

The message inside input() is called a prompt. It helps the user know what to type.

Converting input

If you need a number, convert the input using int() or float(). 

age = int(input("Enter your age: "))
print(age)
price = float(input("Enter a price: "))
total = price + 5.0
print(total)
first = input("First name: ")
last = input("Last name: ")
full_name = first + " " + last
print(full_name)

Tip: Always convert if you plan to do math. Text and numbers are different types.

Printing multiple values

print() can show more than one value. Python adds spaces between them automatically. 

name = "Alex"
age = 20
print("Name:", name, "Age:", age)

4. Conditional Statements

Why programs make decisions

Programs often choose different actions based on information. This is like deciding to bring an umbrella only if it is raining.

if, elif, else

Use if to check a condition, elif to check another, and else for the final fallback. 

temperature = 30
if temperature > 30:
    print("Hot")
elif temperature >= 20:
    print("Warm")
else:
    print("Cool")

Only one branch runs. Python stops after it finds the first true condition.

Indentation matters

Python uses indentation (spaces) to group lines. The indented lines belong to the condition above them. 

if True:
    print("This line is inside the if")
print("This line is outside")

Comparison operators

  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to

Logical operators

  • and: both conditions must be true
  • or: at least one condition is true
  • not: flips true to false (and false to true)
age = 22
has_id = True
if age >= 18 and has_id:
    print("Entry allowed")
is_raining = False
if not is_raining:
    print("No umbrella needed")

Example

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
number = int(input("Enter a number: "))
if number % 2 == 0:
    print("Even")
else:
    print("Odd")

  • Ask the user for their score and print "Pass" if it is 60 or higher, otherwise print "Try again".

Expected Output: Messages that match the conditions you tested.

5. Loops

Why repetition is needed

Loops let you repeat actions without writing the same code over and over. This saves time and reduces mistakes.

for loop

A for loop repeats a set number of times. It often uses range() to create a sequence of numbers. 

for i in range(5):
    print(i)
for i in range(1, 4):
    print("Round", i)
names = ["Alex", "Mia", "Sam"]
for name in names:
    print("Hello", name)

range(start, stop, step) lets you control the numbers. 

for i in range(2, 11, 2):
    print(i)

while loop

A while loop repeats while a condition stays true. 

count = 1
while count <= 3:
    print(count)
    count += 1
password = ""
while password != "python":
    password = input("Enter the password: ")
print("Correct!")
total = 0
number = 1
while number <= 5:
    total = total + number
    number += 1
print(total)

6. Data Sequences

Lists

A list is a collection of items kept in order. It is like a shopping list: you can add, remove, and check items by position.

Indexing: List positions start at 0, not 1. The first item is at index 0. 

colors = ["red", "green", "blue"]
print(colors[0])
colors.append("yellow")
print(colors)
colors.remove("green")
print(len(colors))
colors.insert(1, "purple")
last_color = colors.pop()
print(colors)
print(last_color)
print("red" in colors)

Common list tools:

  • append(): add an item to the end
  • remove(): remove a specific item
  • insert(): add an item at a position
  • pop(): remove and return an item (default is last)
  • len(): get how many items are in the list
  • in: check if an item exists

Lists keep order, so indexing and looping through them is very common in beginner programs.

Tuples

A tuple is like a list, but it cannot be changed after it is created. Use tuples when the values should stay the same. 

point = (3, 5)
print(point[0])
days_in_week = (7, "days")
print(days_in_week)
scores = (10, 20, 10)
print(scores.count(10))
print(scores.index(20))

Tuples are useful for fixed data, like coordinates or days in a week.

Sets

A set is a collection of unique values. It automatically removes duplicates. 

fruits = {"apple", "banana", "apple"}
print(fruits)
fruits.add("orange")
print("banana" in fruits)
fruits.discard("banana")
more_fruits = {"pear", "orange"}
fruits.update(more_fruits)
print(fruits)

Sets do not keep order and do not use indexes.

  • add(): add one item
  • remove() / discard(): remove an item
  • update(): add many items from another set
  • in: check if an item exists

Dictionaries

A dictionary stores data in key-value pairs. You use a key to find its value, like looking up a word in a dictionary. 

student = {
    "name": "Alex",
    "age": 20
}
print(student["name"])
student["grade"] = "A"
print(student)
print(len(student))
print(student.get("name"))
print(student.get("city", "Unknown"))
for key in student.keys():
    print(key)
for value in student.values():
    print(value)
for key, value in student.items():
    print(key, ":", value)
  • keys(): get all keys
  • values(): get all values
  • items(): get key-value pairs
  • get(): safely get a value
  • in: check if a key exists
